Introdução

Não existem motivos para programação sem os requisitos funcionais na informação.
A camada Gerenciadora de Banco de Dados Relacional SQLite está disponível junto às aplicações para armazenamento local ou como recurso de espelhamento na nuvem. A aplicação iDBLite é um Ambiente Visual para o SQLite gratuito e de código aberto; o iDBLite está disponível para criar estruturas de base de dados e realizar manipulações para: popular, atualizar, modificar e excluir, operações vitais para o DB Design no desenvolvimento de aplicativos. Desenvolvido como Prova de Conceito para o estudo da dissertação do Mestrado Profissonal em Desenvolvimento de Sistema no Centro de Informática da Universidade Federal de Pernambuco.
O estudo acadêmico garante que a solução iDBLite é funcionalmente completa, atende aos requisitos e restrições do SQLite com o uso da interface visual.
A solução proposta é livre para uso, estudos ou contribuições.
