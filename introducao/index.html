Introdução

https://mobirise.com/

As soluções informatizadas atuais, embarcadas e cliente-servidor, exigem os requisitos funcionais de persistências de dados.

A camada Gerenciadora de Banco de Dados Relacional SQLite está disponível junto às aplicações para armazenamento local ou como recurso de espelhamento na nuvem. A aplicação iDBLite é um Ambiente Visual para o SQLite gratuito e de código aberto; o iDBLite está disponível para criar estruturas de base de dados e realizar manipulações para: popular, atualizar, modificar e excluir, operações vitais para o DB Design no desenvolvimento de aplicativos. Desenvolvido como Prova de Conceito para o estudo da dissertação do Mestrado Profissonal em Desenvolvimento de Sistema no Centro de Informática da Universidade Federal de Pernambuco, a solução iDBLite mostra-se funcionalmente completa e atende aos requisitos e restrições do SQLite. A solução proposta é livre para uso, estudos ou contribuições.
